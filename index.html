설치가 차단될 때 (SmartScreen · 스마트 앱 컨트롤)
  1. "Windows의 PC 보호" 파란 창이 뜨면 추가 정보 → 실행을 누르세요.
  2. 스마트 앱 컨트롤이 차단하는 경우: Windows 보안 → 앱 및 브라우저 컨트롤 → 스마트 앱 컨트롤 설정에서 끄기를 선택하세요. (한 번 끄면 Windows 재설치 전까지 다시 켤 수 없습니다)
  3. 다운로드한 파일 우클릭 → 속성에서 차단 해제가 보이면 체크 후 확인.
참고 사항
  1. 보고서의 영상 프레임 사진 기능은 PC에 ffmpeg이 설치되어 있어야 동작합니다 (없어도 나머지 기능은 정상).
  2. 설치 버전은 시작 메뉴·바탕화면 바로가기를 만들고, 제거는 Windows 앱 목록에서 가능합니다.
